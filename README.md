# -数字车间大数据监控系统
数字车间大数据监测系统是一个具有数据采集，存储，分析功能的系统。该系统负责接收和解析数字车间底层物联网系统的实时数据流， 存储车间日常运行数据，提供数据查询接口，最后提供可视化，核心为内部的机器学习算法判断机床运行状态、加工工件质量优化预测。
